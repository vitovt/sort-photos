# Сортувальник фото та відео

Зручна настільна утиліта, що допомагає швидко переглядати фото/відео та розкидати їх по різних теках. Працює у вікні Tkinter: зліва бачите попередній перегляд, знизу — підказки з гарячими клавішами.

## Можливості

- Перегляд фото та відео без втрати якості.
- Переміщення **або** копіювання файлів (опція `--mode move|copy`).
- Будь-яка кількість цільових тек (мінімум дві). Для них автоматично створюються гарячі клавіші `1…9`, далі `A…Z`.
- 10 режимів сортування (назва, натуральний порядок, дата створення/зміни, розмір; прямий або зворотний порядок).
- Відео на базі VLC: звук, пауза (пробіл), перемотка на 5 с (`←`/`→`), показ поточного часу та загальної тривалості.
- Фільтр `--filetypes photo|video|all` дає змогу працювати лише з фото, лише з відео або з усіма файлами одразу (за замовчуванням `all`).

## Вимоги

1. **Python 3.9+** (вбудовані модулі + `Pillow` вже у складі).
2. **Tkinter** (є у стандартній поставці CPython).
3. **VLC + python-vlc** для повноцінного відеоплеєра:
 ```bash
 # Linux (APT-приклад)
 sudo apt install vlc
 pip install python-vlc
 ```
 Якщо VLC недоступний, відео показуватимуться як заглушка без відтворення.

## Запуск

```bash
python sort-photos.py \
[--mode move|copy] \
[--sort <режим>] \
[--filetypes photo|video|all] \
<вихідна_тека> <тека_1> <тека_2> [тека_3 ...]
```

### Приклади

```bash
# Перемістити фото у дві теки, сортування за назвою (за замовчуванням)
python sort-photos.py ~/УсіФото ~/Фото_Сім’я ~/Фото_Друзі

# Копіювати, натуральне сортування
python sort-photos.py --mode copy --sort natural ~/УсіФото ~/album1 ~/album2 ~/album3

# Працювати тільки з відео
python sort-photos.py --filetypes video ~/Media ~/mp4_Я ~/mp4_ВідеоБлог

# Сортувати за датою створення, нові спочатку
python sort-photos.py --sort created-rev "D:\AllMedia" "D:\Київ" "D:\Львів"
```

## Керування у вікні

- `1`, `2`, `A`, … — перемістити/скопіювати у відповідну теку (див. підказку внизу).
- `S` — пропустити файл.
- `Q` — завершити програму.
- **Під час відео**:
  - `Space` — пауза/продовжити.
  - `←` / `→` — перемотка на 5 секунд назад/вперед (навіть після завершення ролика).
  - Статусна стрічка показує поточний час та загальну тривалість.

## Поради

- Перед запуском перевірте, що вихідна тека існує, а цільові — бажані назви (програма створить їх, якщо їх ще немає).
- Якщо хочете спочатку “тестово” розкидати файли, використовуйте `--mode copy`, щоб оригінали залишилися на місці.
- Відео з нестандартними кодеками теж працюватимуть, якщо їх відтворює VLC.
